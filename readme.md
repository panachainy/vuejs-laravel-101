# Vuejs-Laravel-101

## Prerequire

Install

```sh
brew update
brew install php
brew install composer
```

Verify

```sh
composer -V
```

## Initial project

### Prerequire init project

```sh
composer global require laravel/installer
```

### Command

#### Create project

```sh
composer create-project --prefer-dist laravel/laravel blog
```

## Getting started

composer update

composer install

### Serve project

```sh
php artisan serve
```

### Migrate

```sh
php artisan migrate
```
